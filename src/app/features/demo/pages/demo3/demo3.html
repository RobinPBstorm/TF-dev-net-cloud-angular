Les pipes

Les pipes "classiques"

Les pipes nous permettent de formatter des informations venant de notre logique.

{{ menu | uppercase }}

On peut aussi formatter des dates:

{{ today | date: 'd/M/yy' }}

{{ today | date: 'longDate':'+0100':'ja' }}

{{ today | date: 'longTime':'+0100':'fr' }}

Liste des formats pour les dates

On peut aussiformatter des monnaies:

{{ 123456.789 | currency:'EUR':'symbol':'1.2-2':'fr' }}

Les pipes custom

Nous pouvons créer nos propres pipes.

Pour ce faire, nous allons créer le pipe avec la commande:

ng generate pipe nom-dossier/nom-pipe

Une fois notre pipe crée, on modifie la méthode transform pour collecter à la mise en forme que l'on souhaite

{{ maVariable | snakeCase }}
