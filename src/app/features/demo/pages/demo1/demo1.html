Mon premier composant

Comment le générer ?

On peut générer un composant angular en utilisant la commande via angular/cli:

ng generate component nom-dossier/nom-composant

ou ng g c nom-dossier/nom-composant

Le composant sera créer dans app dans les dossiers éventuellement spécifiés

Qu'est-ce qui le compose?

Il est composé de minimum de 3 fichiers:

  • un fichier .ts qui contiendra la logique

  • un fichier .html qui contiendra le template

  • un fichier .scss qui contiendra le style
