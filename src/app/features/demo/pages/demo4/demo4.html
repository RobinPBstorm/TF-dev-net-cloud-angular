Les directives

Component Directives

NgStyle

Cette directivs nous permet de jouer avec le style d'une balise

Pour l'utiliser, on doit importer NgStyle du CommonModule

Par exemple

NgClass

Cette directives permet ajouter ou retirer des classes d'une balise

Pour l'utiliser, on doit importer NgClass du CommonModule

Par exemple: il fait {{temperature}} °C aujourd'hui

NgModel

Comme vu dans les bindings, cette directives permet d'associer une propriété de notre logique à notre template

L'association va dans les 2 sens

Par exemple: faisons notre thermostat

Pour l'utiliser, on doit importer NgModel du FormsModule

Structural Directives

@ if / Conditionel

Avec cette directive, on gérer l'affichage de certaines balises en fonction d'une condition

@if (name.length >= 1) {

Bonjour {{name}}

} @else {

Veuillez vous connecter

}

@ for / la boucle

Cette directive nous permet de boucler sur un objet itératif pour pouvoir réaliser un template pour chacun des éléments

    @for (fruit of fruits; let i = $index; track i) {

  • {{fruit}}

    • } @empty {

    Il n'y a pas de fruit

    }

@ switch

Cette directive reste dans la même idée des conditions.

Et comme le switch en js, on vérifie pour chaque case une et une seule valeur

@switch (textColor) { @case ('red') {

Les texte est affiché en rouge

} @default {

La couleur définie n'a pas été listé

} }
